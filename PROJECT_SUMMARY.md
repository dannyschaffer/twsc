## Troubleshooting Common Issues

### Images Not Loading:
1. Check file paths (case-sensitive)
2. Verify images exist in /images folder
3. Check file extensions (.jpg vs .jpeg)

### Mobile Menu Not Working:
1. Verify script.js is loaded
2. Check browser console for errors
3. Test on different mobile devices

### Forms Not Submitting:
1. Update form action URLs
2. Connect to backend service
3. Test email delivery

### Slow Loading:
1. Compress images
2. Use WebP format
3. Enable lazy loading
4. Minimize CSS/JS
5. Use CDN
